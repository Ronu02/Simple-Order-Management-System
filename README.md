Simple Order Management System

A web application where users can register, log in, and manage orders.

Features
1. User Authentication (Login, Logout)
2. Order Management (Create, Read, Update, Delete)
3. SQLite Database for simplicity

Setup Instructions

1. Clone the repository

    git clone

    https://github.com/Ronu02/Simple-Order-Management-System

2. Create a virtual environment
   
3. Install dependencies

   pip install django

4. Run migrations

   python manage.py migrate

5. Create a superuser (for admin access)

   python manage.py createsuperuser

6. Start a development server

   python manage.py runserver
